# 💼 Job Card UI (React + Vite)

A modern and responsive **Job Listing Card UI** built using **React (Vite)**.  
This project displays job openings from top tech companies like Google, Amazon, Meta, Apple, etc.

---

## 🚀 Features

- 📄 Display job listings in card format  
- 🏢 Company logo integration  
- ⏱️ Shows date posted (e.g., 5 days ago)  
- 🏷️ Tags like Full-time, Part-time, Junior, Senior  
- 💰 Pay per hour display  
- 📍 Location info  
- 🔖 Save button with icon (Lucide React)  
- ⚡ Fast performance using Vite  

---

## 🛠️ Tech Stack

- React.js  
- Vite  
- CSS  
- Lucide React (for icons)
